# HackerNews
## Тестовое задание
### Реализовано:
+ ввывод 15 новосетй с возможностью показа большего количества
+ обновление списка каждые 30 секунд, а также возможность обновить вручную
+ фильтрация новостей
+ страница с комментариями древовидной структуры для каждой новости

### Стек:
+ React
+ TypeScrip
+ CssModules
+ React Router
